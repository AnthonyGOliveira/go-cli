
# Changelog


## [Nova versão] - 2025-07-26
### Adicionado
- Comando `hello` para exibir mensagem de saudação personalizada (flag --name). Utiliza `urfave/cli` e `fmt`.
- Comando `ip` para buscar e exibir IPs de um host informado (flag --host). Utiliza `urfave/cli`, `net` (LookupIP) e `log`.
- Comando `server` para listar servidores de nomes (DNS) de um host informado (flag --host). Utiliza `urfave/cli`, `net` (LookupNS) e `log`.
- Uso dos pacotes: urfave/cli, net, log, fmt.
